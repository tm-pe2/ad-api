# ad-api

## Requirements

- [node](https://nodejs.org/en/)
- [yarn](https://yarnpkg.com/getting-started/install)

## Installation

```sh
yarn install
```

Create `.env` file in root with content:

create a [mailtrap](https://mailtrap.io) account and get a login credentials

```md
MAILSERVICE_USER=""
MAILSERVICE_PASS=""
```

## Running

```sh
yarn serve
```
